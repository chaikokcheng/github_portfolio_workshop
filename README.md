# GitHub Portfolio Workshop

A hands-on workshop for CS students to build a standout GitHub profile README from scratch. Learn Markdown, badges, stats widgets, and more — all in one session.

By: Chai Kok Cheng

---

## What You'll Learn

- Write and format content using **Markdown**
- Create the special GitHub profile repository
- Add **skill badges** and **tech stack icons**
- Embed **GitHub stats**, **streak counters**, and **trophies**
- Set up a **snake animation** on your contribution graph
- Deploy a simple site with **GitHub Pages**

## Prerequisites

- A [GitHub](https://github.com) account
- A web browser (everything is done on github.com — no local tools needed)

---

## Quick Start

1. **Read Module 0** to get comfortable with Markdown syntax.
2. **Follow Module 1** to create your profile repo and build your README section by section.
3. **Use Module 2** to pick the badges and icons that match your tech stack.
4. **Reference Module 3** for a complete copy-paste template.
5. Set up **GitHub Pages** (Module 98) or the **Snake Animation** (Module 99).
